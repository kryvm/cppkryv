Проект по структурам в си:

struct Date - включает в себя параметры дня, месяца и года в формате int.
Имеет функции :
input - ввод с проверкой полученных данных
print - вывод даты
Next - получение следующего дня от дня в параметре
Previous - аналогично с прошлым днём
daysfromzero - количество дней от нулевой даты
AddDays - добавление дней к дате
SubtractDays - аналогично вычитание дней
AddDate - сумма дат
SubtractDate - разница дат
daysbetween - кол-во дней между датами
WhichDay - определение и вывод дня недели по дате
Comparision - сравнение какая дата позже другой
struct Time - структура времени включает в себя параметры дней, часов, минут, секунд в формате инт.
inputTime - функция ввода времени с проверкой данных
printTime - вывод времени
daysintime - получения кол-ва дней во времени
datetotime - сведение даты ко времени
addtime - добавить два времени 
subtracttime - аналогично отнять
addtimetodate - добавить время к дате
Subtracttimefromdate - аналогично отнять
ComparisionTime - сравнить два времени

Проект с библиотекой time.h включает в себя тот же список функций с такими же названиями, но разница в отсутствии структур.
Их заменяет библиотека time.h и внутренние значения элементов имеющихся там структур.
tm_mday - день в дате, tm_mon - месяц даты, tm_year - год даты, tm_yday - дни во времени, tm_hour - часы во времени, tm_min - минуты, tm_sec - секунды.
